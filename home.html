What is Telingo?

    Telingo helps second language learners practice speaking their desired language. Telingo allows users to select a language and a skill level and be placed into a video call with another person who is also trying to learn the same language at around the same skill level so that both users can practice speaking to each other. Telingo also provides users a list of conversation topics, a multilingual dictionary, and a text chat to assist the users if needed.
